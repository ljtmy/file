# hikbench SDF性能测试工具使用指南

# 基本语法

```bash
./hikbench 参数名=参数值 参数名=参数值 ...
```

# 参数说明

|**参数**|**说明**|**默认值**|
| ------| -----------------------------------------------------------------------------------------------------------| ------|
|​**​`asymThreadNum=<num>`​** |非对称算法 (SM2) 的线程数|​`256`|
|​**​`symThreadNum=<num>`​** |对称算法 (SM1/SM3/SM4) 的线程数|​`128`|
|​**​`packetSize=<num>`​** |数据包大小，单位为字节（必须是 16 的倍数）|​`1024`|
|​**​`asymLoops=<num>`​** |非对称算法测试的迭代（循环）次数|​`100`|
|​**​`symLoops=<num>`​** |对称算法测试的迭代（循环）次数|​`10000`|
|​**​`sm4Mode=<value>`​** |SM4 密码/工作模式（1\=ECB, 2\=CBC, 4\=CFB, 8\=OFB，可通过按位或组合）|​`15`|
|​**​`sm1Mode=<value>`​** |SM1 密码/工作模式（1\=ECB, 2\=CBC, 4\=CFB, 8\=OFB, 16\=CTR, 32\=MAC，可通过按位或组合）|​`63`|
|​**​`sm2Op=<value>`​** |SM2 操作类型（1\=签名, 2\=验签, 4\=加密, 8\=解密，可通过按位或组合）|​`15`|
|​**​`testAlg=<value>`​** |待测试的算法（1\=SM2, 2\=SM3, 4\=SM4, 8\=SM1，可通过按位或组合）|​`15`|
|​**​`sm4WithKeyImport=0/1`​**|每次 SM4 操作是否使用密钥导入（0\=否, 1\=是）|​`0`|
|​**​`sm2ExternalKey=0/1`​**|SM2 操作是否使用外部密钥（0\=否, 1\=是）|​`0`|

# 其他选项

|**参数**|**说明**|
| ------| ----------------|
|​ **​`-v, --version`​**|显示版本信息|
|​ **​`-h, --help`​**|显示此帮助信息|

# 示例

```bash
# 测试 SM2 签名+加密，4 个线程，循环 1000 次
# (解析：testAlg=1 代表选择 SM2；sm2Op=5 是因为 签名(1) + 加密(4) = 5)
./hikbench testAlg=1 sm2Op=5 symThreadNum=4 symLoops=1000
```

```bash
# 测试 SM3 哈希，8 个线程，循环 5000 次
# (解析：testAlg=2 代表选择 SM3)
./hikbench testAlg=2 symThreadNum=8 symLoops=5000
```

```bash
# 测试 SM4 ECB+CBC 模式，2 个线程，循环 2000 次
# (解析：testAlg=4 代表选择 SM4；sm4Mode=3 是因为 ECB(1) + CBC(2) = 3)
./hikbench testAlg=4 sm4Mode=3 symThreadNum=2 symLoops=2000
```
